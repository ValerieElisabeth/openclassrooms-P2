///////////////////////////////////////////////////////////////////////////////////////

                        (1) INSTALLATION DE SASS (MAC)

///////////////////////////////////////////////////////////////////////////////////////


A) Rendez-vous sur nodejs.org et téléchargez la LTS de NodeJS. // Cok pour moi
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

1) Une fois installé, ouvrez votre terminal et tapez les commandes suivantes :

            |node -v
            |npm -v
_________________
RÉSULAT attendu :
Le terminal de commande affichant les versions :
a) de Node (v16.14.0)
b) de Npm (8.3.1)

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

2) INSTALLEZ SASS. Tapez les commandes suivantes (préfixe : sudo) + PASSWORD du MAC :

            |npm -g install sass
            |sudo npm -g install sass

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

3) Initialisez un fichier npm package.json :

            |npm init
_______________________
INSTRUCTIONS à suivre :
Valider toutes ces étapes et dire (yes) à la dernière question.
_________________
RÉSULAT attendu :
Un fichier : (package.json) à été créé à la racine du site.

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
3) Vérifiez que sass est bien installé :

            |sass --version
_________________
RÉSULAT attendu :
Le terminal affiche notre version de sass.
1.49.9 compiled with dart2js 2.16.1

----------------------------------------------------------------------------------------------
°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°

        4) SASS est installé, il faut le faire FONCTIONNER.
        
----------------------------------------------------------------------------------------------
°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°

5) Aller dans le fichier :
            |package.json

6) Supprimer la ligne :
            |"test": "echo \"Error: no test specified\" && exit 1")

7) La remplacer par la ligne suivante :
            |"sass": "sass --watch front_end/sass/style.scss:front_end/sass/style.css"

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

8) LANCER SASS avec la commande suivante :

            |npm run sass
_________________
RÉSULAT attendu :
Le terminal indique le contenu du script, puis compile votre main.scss en CSS.
            |Compiled sass/main.scss to sass/style.css

----------------------------------------------------------------------------------------------
°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°

        9) COMPILATION DU FICHIER CSS EN MODE COMPRÉSSÉ :
        
----------------------------------------------------------------------------------------------
°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°°

10) Aller dans le fichier :
            |package.json
-------------------------------------------------------------------

11) A la suite de la ligne du script suivant, après  : (…/style.css)
            |"sass": "sass --watch front_end/sass/style.scss:front_end/sass/style.css"
-------------------------------------------------------------------

12) Ajouter le mot : (compressed)
            |"sass": "sass --watch front_end/sass/style.scss:front_end/sass/style.css compressed"


xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

13) INTERROMPRE l'ancien script, puis le RELANCER pour ACTIVER la compression css.
Dans le terminal, tapez la commande suivante :

            |Ctrl+C

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

13) Relancer le script avec la commande suivante :

            |npm run sass

xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

b) WATCH : Permet de voir les changements en direct.

            |sass --watch style.scss:style.css
           (2) |sass --watch front_end/sass/style.scss:front_end/sass/style.css
